# Send Client a Questionnaire Workflow

This workflow streamlines the process of sending dynamic intake or follow-up questionnaires to clients using Docassemble.

:::info
To begin, navigate to the LegalServer matter profile and click the **Send client a questionnaire** link under the Docassemble Interviews block.
:::

## Step 1: Initial Setup

When the interview launches, you will begin the setup process for the client questionnaire.

![Questionnaire Initial Screen](/img/questionnaire_step1.png)

### Selecting the Questionnaire Type
Docassemble will prompt you to define the scope and type of questions you need the client to answer. Because the workflow is linked to LegalServer, the system already knows the client's contact information and the basic case type.

### Dispatching the Link
Follow the prompts to finalize the questionnaire configuration. Docassemble will generate a secure, unique link for the client and dispatch it to them. As the client fills out the questions, their answers will be securely synced back to the case file.
